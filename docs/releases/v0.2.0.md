# Agenvyl v0.2.0 Technical Preview

This Technical Preview makes room workspaces easier to inspect and safer to
share between parallel agent runs. It also expands the supported local coding
agent harnesses and planning controls.

## What's new

- A unified full-screen Workspace and artifact viewer with a collapsible file
  tree, direct links to exact files and versions, and a compact header for
  switching view modes and file history.
- Rendered and source views for HTML, Markdown, and SVG, plus syntax-highlighted
  source for code and text files. Text encoding is detected automatically and
  can be overridden without changing the original file.
- Faster version navigation from the viewer, including pinned historical
  versions, explicit restore, download, and attachment actions.
- Isolated workspace snapshots for agent runs. Completed changes are merged
  back into the room workspace, while conflicting paths remain visible for
  deliberate resolution.
- Experimental Plan Mode with a versioned `plan.md`, approval tied to an exact
  plan version, and an explicit implementation handoff.
- Native Codex CLI and Claude Code CLI harness support, including an
  authenticated loopback permission bridge for Claude Code. Reasoning effort
  can be configured per persona where the selected harness supports it.
- Refined supervisor setup and lifecycle controls, Windows installation, and
  user documentation.

## Platforms and harnesses

Self-contained archives are provided for Linux x64/arm64, macOS x64/arm64, and
Windows x64. They bundle Node.js 22 and PostgreSQL 17.10.

The Connector supports Codex CLI, Claude Code, OpenCode, Antigravity, and
Hermes. Harness capabilities differ; see the
[capability matrix](../harnesses/capabilities.md) before choosing an agent.

## Preview boundary

This release is unsigned and intended for a trusted single-operator
workstation. It has no service/autostart, automatic update, tray application,
telemetry, or multi-user authorization layer. A room workspace is shared agent
state, not a security sandbox. Rendered HTML and SVG are isolated, but may
execute document content within the preview boundary. Review the
[trust guide](../user-guide/trust-and-security.md) before installation.

## Install

Linux or macOS:

```bash
curl -fsSL https://github.com/riffi/agenvyl/releases/download/v0.2.0/install.sh |
  sh -s -- --version 0.2.0
```

Windows PowerShell:

```powershell
$env:AGENVYL_VERSION = '0.2.0'
irm https://github.com/riffi/agenvyl/releases/download/v0.2.0/install.ps1 | iex
```

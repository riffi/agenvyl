# Harness marks

These local assets are used as monochrome masks; they are never fetched at runtime.

- `hermes.svg` is adapted from the official MIT-licensed [Hermes Agent ACP icon](https://github.com/NousResearch/hermes-agent/blob/main/acp_registry/icon.svg).
- `opencode.svg` is adapted from the official MIT-licensed [OpenCode favicon](https://github.com/anomalyco/opencode/blob/dev/packages/ui/src/assets/favicon/favicon.svg).
- `antigravity.svg` embeds Google's official one-color [Antigravity press asset](https://www.antigravity.google/press).

The product names and marks remain the property of their respective owners.
